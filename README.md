Infinity OS
===========
Copyright (C) 2013 Designture — see LICENSE.TXT

O Infinity OS é um sistema operativo desenvolvido por portugueses, para diversas arquiteturas, mas com principal foco em computadores baseados em x86-64 e ARM. O OS é escrito em Assembly, C e C++. O desenvolvimento do Sistema Operativo está apoiado em três grandes pontos:
* Computação de Elevada Performance - O objectivo é fornecer um sistema operativo com elevada performance, através da utilização de novas técnicas de programação, e o suporte as ultimas tecnologia tais como o processamento paralelo e multi núcleo.
* Design e Facilidade de Leitura - Este é um ponto muito importante. Devemos sempre desenhar o código da forma mais fácil possível para facilitar a manutenção do mesmo.
* Educação - Proporcionar um ambiente de aprendizagem e experiências com programação Assembly, C e C++ de baixo nível, bem como os fundamentos dos sistemas operativos.
